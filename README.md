# dotfiles
Personal macOS related dotfiles

## Installation

### Git-free install

```bash
cd; curl -#L https://github.com/antoineneff/dotfiles/tarball/main | tar -xzv --strip-components 1 --exclude='README.md'
./setup.sh
rm setup.sh
```
